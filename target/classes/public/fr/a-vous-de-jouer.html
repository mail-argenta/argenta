Les 10 questions les plus fréquemment posées

Qu’est-ce qu’un code BIC? Et où puis-je le trouver ?

Le code BIC d’Argenta est ARSPBE22.

Le code BIC (Bank Identifier Code) est le code d’identification de la banque. Chaque banque possède son propre code comportant 8 ou 11 caractères. Le code BIC d’Argenta est ARSPBE22. Le code BIC est complété automatiquement (en sous-jacent) lorsque vous remplissez le numéro IBAN.

Quelqu’un peut-il me voler de l’argent rien qu’en tenant un terminal de paiement à proximité de ma carte ?

Le paiement sans contact est une technologie simple, rapide et surtout, totalement sûre. Le scénario selon lequel un fraudeur pourrait voler de l’argent en tenant un terminal de paiement à proximité de la carte, est improbable, et ce, pour différentes raisons :

  • Le terminal doit se trouver dans un rayon de 4 cm autour de la carte pour que la transaction puisse être effectuée. Le voleur devrait donc se tenir très près du détenteur de la carte.
  • Un enregistrement est obligatoire pour détenir un terminal. En cas de transaction frauduleuse, le fraudeur serait donc immédiat identifié.
  • Des règles de sécurité spécifiques sont d’application pour les paiements sans contact. Pour les paiements sans contact supérieurs à 50 euros, votre code PIN vous est demandé régulièrement pour contrôler votre identité.
  • Vous êtes totalement protégé(e) si quelqu’un, en dépit des règles de sécurité, utilisait votre carte de manière frauduleuse.

Pourquoi ma carte de crédit ne fonctionne-t-elle pas ?

  • Votre carte de crédit est bloquée parce que le solde de votre compte de paiement est inférieur à 0. Apurez le solde au plus vite. Le déblocage de votre carte peut prendre jusqu'à deux jours ouvrables bancaires.
  • Il y a une réservation sur votre carte de crédit. Vous ne pouvez pas utiliser le montant réservé pour faire de nouvelles dépenses.
  • Wordline a bloqué votre carte de crédit à titre préventif en raison d’une suspicion de fraude ou d'abus (dans ce cas, vous avez reçu une lettre sur la prévention de fraude ou un SMS).
  • Vous avez atteint la limite de dépenses de votre carte de crédit. Consultez les opérations effectuées avec votre carte de crédit dans l'app Argenta ou Argenta Banque par Internet.
  • Vous avez introduit 3 codes PIN erronés.
  • Votre carte est arrivée à expiration.
  • Votre carte n'a pas encore été activée.
  • Votre carte n'a pas encore été paramétrée pour être utilisée dans certains pays.
  • Vous avez dépassé la limite pour les retraits d'argent.

Qu’est-ce qu’un numéro IBAN? Et où puis-je le trouver ?

Le numéro IBAN (International Bank Account Number) est le numéro de compte international. En Belgique, il se compose de 16 caractères: 2 lettres pour le code du pays (BE), 2 chiffres de contrôle et les 12 chiffres du numéro de compte national. Dans la Banque par Internet, tous vos comptes sont affichés au format IBAN dans la section « Mes comptes ». Vous pouvez également trouver votre numéro IBAN sur vos extraits de compte.

Pourquoi est-ce que je ne peux pas effectuer d’opérations sur mon compte ?

Certains comptes peuvent uniquement être consultés via la Banque par Internet, car ils présentent des caractéristiques qui ne permettent pas d’effectuer des opérations.

Vous ne pouvez pas effectuer d’opérations sur les comptes suivants :

  • un compte pour lequel une limitation a été enregistrée dans l’agence Argenta;
  • un compte pour lequel il faut plus d'une signature;
  • un compte e-épargne, un compte de croissance, un compte Maxi ou un compte Plus qui est lié à l'abonnement d'un mineur d'âge;
  • un compte bloqué;
  • un compte dont vous avez demandé la clôture;
  • un compte de garantie locative.

Pour les comptes d'épargne réglementés, les opérations autorisées sont définies de manière stricte. Elles sont différentes pour les personnes mariées et les cohabitants légaux, d'une part, et pour les cohabitants de fait et les cotitulaires, d'autre part.

Comment puis-je activer itsme® ?

  1. Téléchargez l’app itsme® dans l’App Store ou Google Play.
  2. Créez votre token d’identification via le site web d’itsme®.
  3. Saisissez votre numéro de GSM et votre adresse e-mail.
  4. Connectez votre lecteur de carte et insérez-y votre carte e-ID.
  5. Lisez et signez le contrat itsme® en saisissant le code PIN de votre e-ID.
  6. Vous recevez un token d’identification (6 caractères) que vous devez saisir dans l’app.
  7. Ouvrez l’app itsme®.
  8. Saisissez votre numéro de GSM.
  9. Saisissez votre token d’identification (6 caractères).
  10. Vous recevez un code de contrôle (5 chiffres) par SMS. Saisissez ce code dans votre app.
  11. Choisissez votre code itsme® à 5 chiffres et confirmez-le. Activez aussi Touch ID, Fingerprint et Face ID si vous le souhaitez.
  12. Votre compte itsme® est créé.

Où puis-je trouver un aperçu de la prime de fidélité que je peux encore acquérir ?

  1. Connectez-vous à Argenta Banque par Internet.
  2. Allez dans « Mes comptes ». Sélectionnez un compte d'épargne et choisissez « Intérêts » en haut de l'écran.
  3. Choisissez « Prime de fidélité ».
  4. Cliquez sur « Télécharger ».  En bas de l'écran, vous trouverez une clause de non-responsabilité concernant cette simulation.
  5. Vous recevrez une simulation des primes de fidélité restant à acquérir dans un fichier PDF.
